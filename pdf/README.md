PDF file generated from LaTeX sources.

After each class, notes are written up and stored as .tex files in the 'ĺatex' folder.
These files are then concatenated and turned into a PDF, which is stored in this folder.
The PDF is usually kept closely in sync with the .tex sources, but if there is any doubt,
the .tex sources are always the latest (i.e., I might forget to update the PDF after
changing the .tex files).
